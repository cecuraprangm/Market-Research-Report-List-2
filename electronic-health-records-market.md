<p><p>Electronic Health Records (EHR) market types can be classified into three categories - Web Based, Client Server Based, and Software as Services market. Web Based EHR systems are accessible through a web browser and do not require any software installation on the user's computer. Client Server Based EHR systems require software to be installed on each computer accessing the system, usually within a healthcare facility. Software as Services market offers EHR systems on a subscription model with remote access via the internet.</p></p>
